## Layout Specifications

### Spacing System
- Section Padding: 80px top/bottom (desktop), 40px (mobile)
- Card Gaps: 32px
- Text Margins: 16px between paragraphs

### Breakpoints
- Mobile: < 768px
- Tablet: 768px - 1024px
- Desktop: > 1024px
- Large Desktop: > 1440px

### Animation Triggers
- Fade in: Elements appear at 80% scroll into viewport
- Slide in: Cards slide from sides at 70% scroll
- Count up: Numbers animate when 60% visible
- Parallax: Hero section at 0.5 speed multiplier

### Grid System
- 12-column grid on desktop
- 6-column grid on tablet
- 4-column grid on mobile
- 24px gutters

---

## Unique Visual Elements

1. **QR Code Pattern Dividers**: Between sections, animated QR code patterns as decorative dividers
2. **Isometric Grid Background**: Subtle 3D grid on tech-focused sections
3. **Gradient Overlays**: Red-to-blue gradients on CTA sections
4. **Glassmorphism Cards**: Semi-transparent cards with backdrop blur
5. **Micro-interactions**: Buttons that expand on hover, cards that lift on hover
6. **Progress Dots**: Scroll progress indicators that change as you move through site
7. **Carnegie Red Accents**: Strategic use of CMU red for emphasis
8. **Monospace Metrics**: Tech-style monospace font for all numbers/stats
